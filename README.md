# Toby's Blog

如果你对使用hexo搭建博客也感兴趣的话，可以参考它的[官方文档](https://hexo.io/)。

1. 你需要安装node.js
2. 安装hexo
3. 配置hexo
4. 预览你的博客

## 自定义命令 【需要Python】

```shell
# 新建博客草稿
python @draft.py

# 发布草稿
python @prepare.py draft

# 本地预览
python @preview.py

# 推送保存
python @push.py

```
